PROPOSAL

Bulletin is a project management web application. Using JavaScript & the Browser DOM to create an interactive interface to plan out your project requirements and keep track of your progress. The application will utiilize local storage to save your personal 'Bulletin' board. The functionaly will closely resemeble the magnetic DOM homework with clicking and dragging rectangles. In addition you will be able to create rectangles with custom text and expand each rectangle for more information about the provided text. These rectangles can also change color to represent their current status.
